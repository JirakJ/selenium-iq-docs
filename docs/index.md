# SeleniumIQ Documentation Hub

Welcome to the official docs for the SeleniumIQ IntelliJ plugin. Use this site to master AI-assisted locator generation, stay aligned with releases, and file documentation updates.

## What You Can Do Here

- **Learn quickly** with onboarding checklists and guided workflows
- **Track releases** via the public changelog synchronized with the plugin repo
- **Escalate issues** through standardized templates for bugs, features, and docs
- **Compare tiers** to understand which premium plan unlocks specific capabilities
- **Troubleshoot fast** using curated fixes for the most common setup snags
